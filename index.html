STORYBOOK

Storybook is a reminiscence therapy app developed as a part of a term-long group project for CMPT 276 at SFU. The goal of the project was to come up with an app that could, in some way, improve the life of someone living with Dementia. The app was written in Java in Android Studio, and features a photo gallery that can cannot be edited or deleted by anyone other than an admin caretaker. The app takes user-uploaded photos and stored in a Cloud Firestore database so it can be accessed remotely. The app also features a tag system and a family log so users can read which people are in each photo.
